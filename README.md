# FUTURE_DS_02
# CUSTOMER CHURN ANALYSIS DASHBOARD

## Objective
Analyze customer churn patterns and identify factors influencing customer retention using the Telco Customer Churn dataset.

## Dataset
Telco Customer Churn Dataset

## Tool Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

## Dashboard Components
- Overall Churn Analysis
- Churn by Contract Type
- Churn by Internet Service
- Churn by Payment Method
- Customer Tenure Analysis

## Key Insights
1. Overall churn rate is approximately 26.5%.
2. Month-to-Month contracts showed the highest churn.
3. Two-Year contract customers had the strongest retention.
4. Electronic Check users recorded the highest churn.
5. Customers in their early tenure months were more likely to leave.
6. Long-term customers showed better loyalty and retention.

## Recommendations
1. Promote annual and two-year contract plans.
2. Encourage customers to switch to Auto-Pay methods.
3. Improve onboarding support for new customers.
4. Offer loyalty rewards for long-term customers.
5. Target high-risk customers with retention campaigns.

## Conclusion
The analysis identified key churn drivers and customer retention trends. Long-term contracts and loyal customers showed lower churn rates, while month-to-month customers were more likely to leave. These insights can help businesses improve customer retention strategies.
